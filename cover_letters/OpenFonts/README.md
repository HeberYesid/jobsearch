# OpenFonts

Place the Lato and Raleway font files here:

- **Lato**: https://fonts.google.com/specimen/Lato
- **Raleway**: https://fonts.google.com/specimen/Raleway

Download the TTF/OTF files and place them in this directory. `xelatex` will load them via `fontspec`.

If the fonts are not found, the cover letter class falls back to the default sans-serif font automatically.
