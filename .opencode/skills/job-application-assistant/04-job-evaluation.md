# Job Evaluation Framework (7 Dimensions)

This file defines the fit evaluation framework used by `/apply`. The structure of `profile/job-evaluation.md` is generated by `/setup`.

## 7 Dimensions

### 1. Skills
Match between posting requirements and candidate's profile skills.
- **Evaluation**: List required skills, match against profile, note gaps.
- **Score**: 0-100 based on percentage of required skills present.

### 2. Experience
Years and relevance of experience vs posting requirements.
- **Evaluation**: Compare years required vs candidate's years. Assess relevance of past roles.
- **Score**: 0-100 based on years match and role relevance.

### 3. Culture
Values, team structure, work style alignment.
- **Evaluation**: Compare company culture signals (from posting/research) against behavioral profile.
- **Score**: 0-100 based on alignment.

### 4. Location
Geography, relocation, remote feasibility.
- **Evaluation**: Check if candidate's location matches or if remote work is feasible.
- **Score**: 0-100 (100 if remote-friendly or same location; lower if relocation required without support).

### 5. Career Alignment
Trajectory direction — does this role move the candidate toward their goals?
- **Evaluation**: Compare role against candidate's stated career direction.
- **Score**: 0-100 based on alignment with trajectory.

### 6. Modality
Remote / hybrid / onsite match.
- **Evaluation**: Compare posting's modality against candidate's preferences.
- **Score**: 0-100 (100 if exact match; lower if mismatch, e.g., candidate wants remote but posting is onsite).

### 7. Salary Band
Conditional on data availability.
- **Evaluation**: If salary data is available in posting or researchable, compare against candidate's band.
- **Score**: 0-100 if data available; "N/A — no data" if not.
- **Note**: This dimension is conditional. Do NOT penalize fit score when salary data is unavailable.

## Deal-Breakers

Deal-breakers are conditions that make the role unacceptable regardless of other scores:
- Location mismatch without remote option
- Salary below candidate's minimum (when data available)
- Modality mismatch (candidate wants remote, posting is 100% onsite with no flexibility)
- Visa/sponsorship required but candidate cannot comply

Deal-breakers surface explicitly. The candidate decides whether to proceed.

## Overall Fit Score

Weighted average of available dimensions. Salary band excluded from average when data unavailable.
